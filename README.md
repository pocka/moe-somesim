# MoE そめしむ

![そめしむロゴ](./docs/logo.svg)

## アプリケーション URL

<https://pocka.github.io/moe-somesim/>

## これは何？

MoE ([Master of Epic](http://moepic.com), 日本の MMORPG) の染色システムをシミュレーションするブラウザアプリです。
まず染色したい装備を選んでから染色に利用する花びらを 1 枚以上選ぶと、ゲーム内と同じように装備の染色を行うことができます。
